# kokkai-summary

国会会議録要約システム(仮)

## motivation

国会会議録は長すぎて、その全文をそのまま読むのは困難。
217回常会を例にとると、10万字越えの会議録が75件あり、全体で3350万字に達していて読むに堪えない。
最近はGemini・ChatGPTをはじめ、要約をAIに行わせる素地が整ってきたので、何とかしてこの字数を減らしたい。
あるいは、誰がどんな主張をしてきているか、それをより簡単に取得するための素地を整えたい。
